** Quiz App using JavaScript

🔧 Technologies Used:
 ✅ HTML – to structure the layout of the quiz application.
 ✅ CSS – for styling and making the UI interactive and responsive helps to make application colorful.
 ✅ JavaScript – to handle quiz logic, to get result by calculating correct answer.

✨ Key Features:
 🔸 Dynamic Questions and Choices rendered using JavaScript.
 🔸 Next Button to proceed to the next question.
 🔸 Score Tracker to show results at the end and if want to start click play again button.
 🔸 as click on answer immediately get response that your answer is correct or incorrect.

💡 Challenges Faced:
 ⚠️ Handling DOM manipulation to dynamically update questions and choices 
 ⚠️ As get response for answer immediatly shows correct or incorrect.
 ⚠️ Ensuring score accuracy and updating it based on selected answers
